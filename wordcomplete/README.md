# wordcomplete
* command line utility to list all words with a common prefix. 
* primarily its to illustrate the use of the trie data structure. 
* this could also be implemented using a single line grep command in bash.

> how to compile: 
    *make* 

> how to run: 
    *./wordcomplete "prefix"* 

> example: 
    ./wordcomplete alpha
